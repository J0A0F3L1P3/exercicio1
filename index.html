Desvios Condicionais Simples

  • EXERCÍCIO 1.

    Faça um programa que leia dois números quaisquer, e escreva o maior deles.

    EXERCÍCIO 2.

    Faça um programa que leia um número inteiro e informe se o número é "PAR" ou "IMPAR".

    EXERCÍCIO 3.

    Faça um programa que leia dois números quaisquer, e escreva o resultado do cálculo do maior divido pelo menor.

    EXERCÍCIO 4.

    Faça um programa que leia a receita e a despesa de uma empresa e apresente se ela está com LUCRO ou com PREJUÍZO.

    EXERCÍCIO 5.

    Faça um programa que leia dois números e apresente o total do produto entre eles. Considere que somente números diferentes de zero serão digitados.

    EXERCÍCIO 6.

    Faça um programa que leia um número e escreva "maior que 100", se o número digitado for maior que 100 e escreva "menor que 100", se o número for menor do que 100.

    EXERCÍCIO 7.

    Faça um programa que leia um código: 1-sinal vermelho, 2-sinal verde. Baseado no código digitado diga para o pedestre que está esperando no sinal: "SIGA" ou "ESPERE".

    EXERCÍCIO 8.

    Faça um programa que leia a quantidade de vitórias e a quantidade de derrotas de um time de futebol. Escrever BOM se o número de vitórias for maior que o número de derrotas, caso contrário escrever RUIM.

    EXERCÍCIO EXTRA.

    Para crédito imobiliário, uma intituição bancária estipulou que a prestação não poderá ultrapassar 30% do salário bruto do funcionário. Crie um algoritmo que solicita o salário e indica se o crédito foi aprovado ou não.

  • Tipos de Variáveis em C#


    As variáveis em C# são usadas para armazenar dados em memória durante a execução do programa. C# é uma linguagem fortemente tipada, o que significa que todas as variáveis devem ter um tipo especificado. Aqui estão os principais tipos de variáveis em C#:

    int: Representa números inteiros. Exemplo: int idade = 25;

    double: Representa números de ponto flutuante com maior precisão. Exemplo: double altura = 1.75;

    float: Representa números de ponto flutuante com menor precisão. Exemplo: float temperatura = 25.5f;

    decimal: Representa números decimais com alta precisão. Exemplo: decimal valor = 10.99m;

    bool: Representa um valor booleano verdadeiro ou falso. Exemplo: bool temCNH = true;

    char: Representa um único caractere. Exemplo: char genero = 'M';

    string: Representa uma sequência de caracteres. Exemplo: string nome = "Maria";

    Além desses tipos primitivos, existem outros tipos avançados, como arrays, estruturas e classes, que permitem armazenar conjuntos de valores ou criar tipos personalizados.
